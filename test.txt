1. Создать index.html с базовой разметкой документа. Создать файл скрипта JS.
2. Подключить файл скрипта JS к документу.
3. В документе создать следующие элементы:
3.1. Поле для ввода имени
3.2. Поле для ввода возраста
3.3. Кнопка Сохранить(<input type='button' value='Send'/>)
3.4. Кнопка Показать
3.5. Блок для вывода Результата
4. Создать следующую логику
4.1. Написать обработку данных из полей. Поле имя должно сохранить свое содержимое в this.name, поле возраст в this.age. Событие должно вызываться при нажатии на кнопку Сохранить
4.2. Написать вывод сохраненных данных в блок Результата по нажатию на кнопку Показать. Вывод должен быть формата:
Имя: <ИМЯ>, Возраст: <ВОЗРАСТ>
